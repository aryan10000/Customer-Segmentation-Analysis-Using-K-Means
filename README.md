# Customer Segmentation Analysis Using K-Means

Customer analytics project using Python and K-Means clustering to identify distinct customer segments based on income and spending behavior. The project uncovers actionable insights that can support targeted marketing strategies and customer retention initiatives.

## Project Overview

Understanding customer behavior is essential for effective marketing and business growth. This project applies unsupervised machine learning techniques to segment customers into distinct groups based on their purchasing patterns and demographic characteristics.

By identifying high-value and low-value customer segments, businesses can optimize marketing campaigns, personalize offers, and improve customer engagement.

---

## Business Problem

Businesses often treat all customers similarly despite significant differences in spending habits and purchasing power.

This project aims to answer:

- Can customers be grouped into meaningful segments?
- Which customer groups generate the highest business value?
- How can businesses target different customer segments more effectively?
- Which customers should receive premium offers and loyalty programs?

---

## Dataset

The dataset contains customer demographic and spending information.

### Features

- Customer ID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1–100)

### Dataset Overview

- 200 customer records
- Customer demographic information
- Income and spending behavior metrics

---

## Project Workflow

### 1. Exploratory Data Analysis (EDA)

Performed:

- Distribution analysis
- Correlation analysis
- Univariate analysis
- Bivariate analysis
- Customer behavior exploration

### 2. Customer Segmentation

Applied:

- K-Means Clustering
- Elbow Method for optimal cluster selection
- Cluster visualization and interpretation

### 3. Cluster Analysis

Identified customer segments based on:

- Annual Income
- Spending Score
- Customer purchasing behavior

---

## Key Insights

### Segment 1: High Income, High Spending
- Premium customers
- High business value
- Suitable for loyalty programs and exclusive offers

### Segment 2: High Income, Low Spending
- Underutilized customers
- Opportunity for targeted marketing campaigns

### Segment 3: Low Income, High Spending
- Highly engaged customers
- Responsive to promotions and discounts

### Segment 4: Low Income, Low Spending
- Budget-conscious customers
- Lower marketing priority

### Segment 5: Average Income, Average Spending
- Core customer base
- Consistent purchasing behavior
